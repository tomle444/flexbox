Flexbox Quick Start Guide

Flexbox is a css display property that has been gaining a lot of attention as Responsive Web Design becoming a popular practice among web designers. You can see examples of Flexbox on sites such as InVision.com

Parent elements can be declared as Flex by assigning display: flex , or inline-flex

In this example, we are putting justify-content: center to center align all of the child elements. Think of it as the equivilent of adding margin: auto text-align: center

To learn more about Flex Box visit CSS Tricks

Flex - Justify Center

.container-1 {
	display: flex;
	justify-content: center
}
.grid-1 {
	background: tomato;
	padding: 5px;
	width: 200px;
	height: 150px;
	line-height: 150px;
	color: white;
	font-weight: bold;
	font-size: 3em;
	text-align: center;
}
